═══════════════════════════════════════════════════════════════════
  CONVENCIONES DE NOMBRES
═══════════════════════════════════════════════════════════════════

1. Todos los nombres en minúsculas
2. Palabras separadas por guiones (-)
3. Prefijos claros según tipo:
   - event-XX-       para eventos
   - button-         para botones
   - item-           para objetos de inventario
   - enemy-          para enemigos
   - state-          para estados

4. Formato: PNG con transparencia
5. Sin espacios ni caracteres especiales
6. Números con dos dígitos (01, 02, 03...)

  INSTRUCCIONES DE USO

1. Crea/coloca cada sprite PNG con el nombre exacto especificado

Una vez colocadas las imágenes en sus carpetas correspondientes,
el juego las cargará automáticamente usando estas rutas.
